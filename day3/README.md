# wttr-cli

[![Python](https://img.shields.io/badge/python-3.8+-blue.svg)](https://python.org)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

> 一行命令，看一眼天气，然后继续写代码。

终端天气极简仪——基于 wttr.in 的零配置命令行天气查询工具。无需 API Key，无需注册，输入城市名即可查看天气。

## Why?

每次想看一眼天气，都要经历：解锁手机 → 打开天气 App → 等待加载 → 查看 → 放下手机 → 重新聚焦代码。这个"上下文切换"至少消耗 30 秒，还打断了心流。

**wttr-cli** 让天气查询留在终端里。一行命令，零摩擦。

## Quick Start

```bash
# 安装
pip install -e .

# 自动检测位置
wttr

# 查询指定城市（支持中文）
wttr 北京

# 查询英文城市名
wttr "New York"

# 3 天预报
wttr 上海 -d 3
```

## Features

- 🌍 **零配置** — 无需 API Key、无需注册、无需登录
- 🏙️ **中英文城市名** — 支持"北京"、"New York" 等
- 📍 **IP 自动定位** — 不带参数自动检测当前位置
- 🎨 **终端美化** — 温度颜色编码、Panel 卡片 + Table 表格
- 📅 **多日预报** — 最多 3 天天气预报
- 📝 **极简模式** — `--simple` 输出纯文本，适合脚本集成

## CLI Reference

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `city` | 城市名（中/英文），省略则自动定位 | 自动定位 |
| `-d, --days` | 预报天数（1-3） | 3 |
| `-s, --simple` | 极简模式，纯文本输出 | 关闭 |
| `--version` | 显示版本号 | — |
